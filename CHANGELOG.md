# SafeHavun Changelog

All notable changes to SafeHavun will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-20

### Added
- Initial release of SafeHavun - Smart Money Crypto Tracker
- Dashboard with portfolio overview and market stats
- Fear & Greed Index integration (Alternative.me API)
- Whale wallet tracking (Etherscan, Blockchain.com)
- CoinGecko price data integration
- Real-time crypto prices and market cap
- Artisan commands for data synchronization
- PWA support for mobile access
- Dark/Light theme toggle

### Technical
- Laravel 11 framework
- MySQL database
- Scheduled tasks for data updates
- RESTful API endpoints
